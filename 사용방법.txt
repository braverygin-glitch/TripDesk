TripDesk V1.0.10.2 사용방법

이번 버전은 Firebase Redirect 로그인 흐름 수정 버전입니다.

PC:
1. GitHub Pages 주소 접속
2. 더보기
3. Firebase 설정 저장
4. 연결 확인
5. Google 로그인
6. 로그인 후 TripDesk로 돌아오면 PC/현재 데이터 업로드

휴대폰:
1. 같은 주소 접속
2. Firebase 설정 저장
3. Google 로그인
4. 클라우드에서 불러오기
5. 실시간 동기화 시작
