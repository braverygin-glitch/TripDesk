# TripDesk

개인용 여행 관리 PWA 프로젝트입니다.

## 현재 버전
V1.0.9 UI Stabilization

## 이번 버전
모바일 사용성을 기준으로 UI를 정리했습니다.

- 여행 카드 버튼 가로 정렬
- 수정/복사/백업/삭제 버튼 한 줄 표시
- 버튼 줄바꿈 방지
- 카드 여백 조정
- 모바일 하단 네비게이션 최적화
- 새 여행/가져오기 버튼 아이콘 추가
- 샘플 여행 안내 문구 정리
