# TripDesk

개인용 여행 관리 PWA 프로젝트입니다.

## 현재 버전
V1.1.0 Firebase Cleanup

## 이번 버전
Firebase 동기화 관리 기능을 정리했습니다.

- 현재 로컬 데이터로 클라우드 덮어쓰기
- 클라우드 전체 초기화
- 이 기기 로컬 데이터 초기화
- 삭제 동기화 개선
- 실시간 동기화 시작 시 클라우드를 현재 로컬 데이터 기준으로 갱신

## 중복 여행 정리 방법

1. 목록에서 남길 여행 1개만 남기고 나머지는 삭제합니다.
2. 더보기 → Firebase 정리 → 현재 로컬 데이터로 클라우드 덮어쓰기를 누릅니다.
3. 휴대폰에서 클라우드에서 불러오기를 다시 실행합니다.
