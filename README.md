# TripDesk

개인용 여행 관리 PWA 프로젝트입니다.

## 현재 버전
V1.3.0 Built-in Firebase Config

## 이번 버전
- Firebase 설정을 앱에 내장
- 휴대폰에서 firebaseConfig를 다시 붙여넣지 않도록 수정
- 첫 화면 Firebase 동기화 카드 단순화
- 더보기 Firebase 동기화 화면 단순화
- 일반 사용자는 Google 로그인만 하면 됨

## 사용 방식
처음 한 번:
1. Google 로그인
2. PC에서 현재 데이터 업로드 또는 휴대폰에서 클라우드 불러오기
3. 실시간 동기화 시작
