# TripDesk

개인용 여행 관리 PWA 프로젝트입니다.

## 현재 버전
V1.0.10.1 Firebase Redirect Login Fix

## 이번 패치
- Google 로그인 방식을 팝업 방식에서 리디렉션 방식으로 변경
- GitHub Pages, iPhone Safari, PWA 모드에서 팝업 차단 문제 완화
- Firebase 로그인 오류 메시지에 실제 오류 코드 표시

## 사용 방법
1. GitHub Pages 주소에서 TripDesk 실행
2. 더보기
3. Firebase 동기화
4. Google 로그인
5. Google 로그인 화면으로 이동
6. 로그인 후 TripDesk로 자동 복귀

## 주의
